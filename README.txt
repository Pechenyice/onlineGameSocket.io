to play lite version of game with guest enter only (liteVersion):

1)install node.js
2)install modules from package.json (if you need)
3)start server (node main.js)
4)go to localhost:3000/
5)play with your friends or play with yourself in different tabs

to play and test full version of game (master):

1)install node.js
2)install modules from package.json (probably you will need to install
mysql2 only (npm install mysql2) )
3)install MySQL and create database users and table players there
3.5)change 'YOUR_PASSWORD' in code with your MySQL password
4)start server (node main.js)
5)go to localhost:3000/
6)play with your friends or play with yourself in different tabs
